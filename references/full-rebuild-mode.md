# Full-rebuild mode

## Objective

Treat the source deck as a verified business-content library and construct a clearer narrative with traceable evidence.

## Planning sequence

1. Extract all audience content, charts, tables, numbers, images, notes, and sources.
2. State the audience, decision, and one-sentence deck thesis.
3. Group source evidence into chapters.
4. Assign each output page one communication job and one core claim.
5. Apply mandatory split and permitted merge rules.
6. Select a composition family, then a T01-T23 layout.
7. Record `sourceSlides`, `retainedEvidence`, and `removedOrCondensed` for every output page.

## Default narrative arc

Use only when supported by the material:

```text
thesis / opportunity
context / problem
capability / method
evidence / examples
boundaries / risks
implementation / action
closing / contact
```

Do not force this arc onto a report whose evidence requires a different structure.

## Rewriting boundary

Rewrite for clarity and compression, not invention. Preserve every factual constraint. Mark unsupported gaps instead of filling them with plausible-sounding claims.

## Editorial pacing

Alternate analytical evidence with lower-density synthesis, chapter, quotation, or image-led pages. Do not add breathing pages that contain no useful audience value.
